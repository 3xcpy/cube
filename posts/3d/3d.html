How it's made

This site is made using three.js and plain HTML & CSS in a few hours. It's a proof of concept for 3D technology on the web. With more and more websites using 3D, it's important to understand how to use it.

There are a lot of resources on how to make a website like this. I used a tutorial by the YouTuber Fireship

It's not that difficult to make a website like this. You just need to know how to use three.js and how to use HTML & CSS. If you follow a tutorial and modify the result, you can make something original and unique.
